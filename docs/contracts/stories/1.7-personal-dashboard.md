# User Story: 個人化使用者儀表板

**Epic:** 1
**Story:** 1.7

## 標題
作為使用者，我希望在儀表板上看到個人化的資訊

## 使用者故事 (User Story)
作為一名使用者，我希望能有一個個人化的儀表板，這樣我就可以快速概覽我參與的讀書會活動和管理我的個人資訊，而不是看到通用的假資料。

## 驗收標準 (Acceptance Criteria)
1.  儀表板必須顯示當前登入使用者自己的資料，而不是假的靜態資料。
2.  儀表板需要顯示使用者參加的讀書會總數。
3.  儀表板需要列出使用者參加的所有讀書會的名稱。
4.  使用者個人資訊（例如：姓名、頭像）應顯示在儀表板的右半部，並以垂直方式排列。
5.  儀表板上應包含以下快捷操作按鈕：
    *   創建讀書會
    *   搜尋讀書會
    *   帳號設定
6.  儀表板上應移除目前顯示「閱讀書籍」的區塊。

## 備註 (Notes)
- 此故事旨在提升使用者體驗，提供個人化且相關的資訊。
- UI/UX 細節將在開發過程中與設計師進一步確認。
